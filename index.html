Every HTML code are enclosed by tags and every tags have thier default settings.

Types of HTML tags includes:
  1. Contianer tags:it has both opening and closing tags e.g < p > < / p > ,< h1 > < / h1 > etc.
  2. Empty tags/Self closing tags: it doesn't have a closing tags e.g < br >, < hr > etc.
  3. Comment tags: These are used for reference or documentation purposes, i.e it should not be interpreted e.g < !-- Hello -->

ATTRIBUTES: Attributes are used to change the default settings of a tag. It can only be used with the opening tags and it must have a name and an assigned value.
